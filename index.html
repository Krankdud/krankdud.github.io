WA FerryTime

Odyssey SpeedTech

A website created to archive videos of tricks and glitches in the game Super Mario Odyssey. Clips can be added and found through tags, or via search. The website was built in Python 3 using the Flask framework. The videos are embedded YouTube, Twitter, or Twitch videos that are fetched via oEmbed. Bootstrap is used for the UI.

NanoSplit

A web-based timer for speedrunners that is designed for mobile devices. NanoSplit allows users to import data from the popular LiveSplit timer, and also export times in LiveSplit's format. This was built using TypeScript and React.

Talkatoo!

A silly project inspired by watching @SpikeVegeta playing through Super Mario Odyssey with the restriction of only obtaining the Moons that the bird, Talkatoo, tells you to grab. This website generates a list of Moons to obtain as you are playing through Super Mario Odyssey. It is built using the Vue.js framework and Bootstrap for the UI.

Mega Man Zero 2 Practice Hack

Contributed some ARM assembly code to @Ajarmar_'s Mega Man Zero 2 practice hack. I added code to set the player's equipment to the appropriate equipment that you would have in a hard mode speedrun, and refactored how ROM regions are defined to make it easier to modify. It was a fun project and a way to learn about the inner workings of the Gameboy Advance.

Speedrunning

Awesome Games Done Quick 2019

I participated in the Games Done Quick speedrunning marathon in January 2019 to help raise over 2.4 dollars million for the Prevent Cancer Foundation! I had two runs in the marathon, a Mega Man Zero 2 Hard Mode speedrun and a N++ Coop speedrun with my good friend AND4H. Both runs went extremely well, and I recommend watching both of them! The Mega Man Zero 2 run was later featured on GDQ's podcast, The Sprint, where the run was broken down by Mithical9, who provided commentary in the live run.

JRTA 4

In November 2017, I participated in the JRTA 4 marathon to help raise money to bring four Japanese speedrunners to Awesome Games Done Quick 2018. During the event, I had two runs: Dustforce and Gunstar Heroes Coop (with AND4H). We successfully raised the money required to bring the four to AGDQ 2018, and the excess money was donated to Doctors Without Borders.

Other projects

Nuclear Zone

A custom level pack created for the game Dustforce. I created 13 levels and a nexus that connects each of the levels together over several months. I wanted to explore a post-apocalyptic style that I had created in a previous level that I made. There's some stuff I would like to change, but overall it turned out pretty well and the project was well received by the community. The recommended way to play it nowadays is through the Community Nexuses in Dustmod, and it even has a revamped nexus thanks to @gdnght_grl!
